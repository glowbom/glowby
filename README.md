# Glowby OSS

Glowby OSS is being rebuilt around a local-first coding agent stack. This repository will soon contain the Go backend and React app that power project refinement, ChatGPT login, OpenCode integration, and live agent runs on your own machine.

While that transition is happening, the older Glowby code is being moved into `legacy/` for reference. The goal is simple: make Glowby easier to clone, run, and extend as an open source agent workflow for real software projects.
